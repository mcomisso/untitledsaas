# Task Completion Checklist

## Before Committing Changes

1. **Test locally** - Run local server and verify changes in browser
2. **Check responsive design** - Test at different viewport sizes
3. **Verify visual effects** - Ensure particle effects and animations work
4. **Check console** - No JavaScript errors
5. **Cross-browser check** - If possible, test in multiple browsers

## No Automated Checks
This project has no:
- Linting tools
- Type checking
- Automated tests
- Build process

All verification is manual through browser testing.

## Git Workflow
1. `git status` - Review changes
2. `git diff` - Check specific changes
3. `git add .` - Stage changes
4. `git commit -m "descriptive message"` - Commit
5. `git push` - Push to remote (triggers GitHub Pages deployment)
